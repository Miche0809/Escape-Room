🧩 Desafio A: Operações Básicas

Determine o valor final da variável resultado:

Qual será o valor final de contador?

Biblioteca dos Mortos

Livros se abrem sozinhos... Páginas voam pelo ar como borboletas fantasmagóricas. O fantasma do bibliotecário sussurra: "Complete ambos os enigmas condicionais..."

📚 Desafio A: Condicionais

O que será exibido na tela?

programa { funcao inicio() { inteiro idade = 16 se (idade >= 18) { escreva("Maior de idade") } senao se (idade >= 13) { escreva("Adolescente") } senao { escreva("Criança") } } }
🎯 Desafio B: Comparações

Qual será o valor de resultado?

programa { funcao inicio() { inteiro x = 8 inteiro y = 12 inteiro resultado se (x > y) { resultado = x + y } senao { resultado = y - x } escreva("Resultado: ", resultado) } }

Laboratório Maldito

Tubos de ensaio borbulham... Esqueletos se movem nas prateleiras. O espírito do professor ri: "Resolva meus experimentos de repetição!"

🔬 Desafio A: Loop Simples

Quantos números serão exibidos?

programa { funcao inicio() { para (inteiro i = 2; i <= 10; i = i + 2) { escreva(i, " ") } } }
🧪 Desafio B: Acumulador

Qual será o valor final de soma?

programa { funcao inicio() { inteiro soma = 0 para (inteiro i = 1; i <= 4; i++) { soma = soma + i * 2 } escreva("Soma: ", soma) } }
🔒
Cadeado Trancado

Laboratório Digital Assombrado

Computadores ligam sozinhos... Telas piscam com códigos sinistros. Uma voz eletrônica ecoa: "Complete meus algoritmos de vetores!"

💾 Desafio A: Soma de Vetor

Qual será o valor da soma?

programa { funcao inicio() { inteiro numeros[5] = {10, 20, 30, 40, 50} inteiro soma = 0 para (inteiro i = 0; i < 5; i++) { soma = soma + numeros[i] } escreva("Soma: ", soma) } }
🔍 Desafio B: Busca no Vetor

Qual será o valor de posicao?

programa { funcao inicio() { inteiro valores[4] = {5, 10, 15, 20} inteiro posicao = -1 para (inteiro i = 0; i < 4; i++) { se (valores[i] == 15) { posicao = i } } escreva("Posição: ", posicao) } }
🔒
Cadeado Trancado

Cantina dos Horrores - Desafio Final

Panelas fervem com ingredientes misteriosos... O fantasma da merendeira surge: "Este é o teste final! Resolva ambos os enigmas de funções para escapar!"

🍕 Desafio A: Função Simples

Qual será o resultado exibido?

programa { funcao inteiro calcular(inteiro x, inteiro y) { retorne x * y + 5 } funcao inicio() { inteiro resultado = calcular(3, 4) escreva("Resultado: ", resultado) } }

Qual será o valor de total?

programa { funcao inteiro fatorial(inteiro n) { se (n <= 1) { retorne 1 } senao { retorne n * fatorial(n - 1) } } funcao inicio() { inteiro total = fatorial(4) escreva("Total: ", total) } }
